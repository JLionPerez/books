# :books: Book Search App

**Live Demo:** https://booksearchgalore.netlify.app/

A React application that allows users to search for books using the Google Books API and view results dynamically in real-time.

## :pencil: Features

- Search for books using any terms
- Fetch and display book information from an API
- Dynamic UI updates based on user input
- Interactive user interface built with Next.js

## :computer: Tech Stack

- Next.js (React)
- JavaScript
- HTML/CSS
- External API integration

## :gear: How to Run Locally

1. Clone the repository:  
```git clone https://github.com/JLionPerez/books.git```

2. Navigate into the project folder:  
```cd books```

3. Install dependencies:  
```npm install```

4. Run the development server:  
```npm run dev```

5. Click on the localhost link or type ```http://localhost:3000``` into your browser.
